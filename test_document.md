# Test Document - Knowledge Base

This is a test document to validate the RAG (Retrieval Augmented Generation) system.

## Important Information

The Basile_IA_Orch project is an AI agent orchestrator that uses LangGraph to manage complex workflows.

### Main Features

1. **MCP Tools**: Integration with Model Context Protocol for external tools
2. **RAG**: Retrieval Augmented Generation using Weaviate as vector store
3. **Agent Collaboration**: Multiple agents can collaborate on tasks
4. **Resilience**: Configuration for retry, fallback, timeout and human-in-the-loop

### Technologies Used

- **Backend**: Python, FastAPI, LangGraph
- **Frontend**: Vue.js, Vuetify
- **Database**: PostgreSQL
- **Vector Store**: Weaviate
- **Embeddings**: sentence-transformers-paraphrase-multilingual-MiniLM-L12-v2 (local)
